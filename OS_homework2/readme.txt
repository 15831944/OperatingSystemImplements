Operating system HomeWork2

Guide:
You might not have curses header file

please type in "sudo apt-get install ncurses-dev" to install the curses library 

use the following command or the makefile in the folder to compile the c file
"gcc oshw2.c -o outputname -lpthread -lcurses"

this program is mainly done with pthread and curses library
pthread to control logflow, display output

after compiling the c file, just input"./outputname" to play the game

Enjoy!

info:
before enter the game, you need to key in the difficulty of the game

this value varies from 1 to 10, lower is harder selection

After keying in the value, the river and frog game will be started!

Have fun!
